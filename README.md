Tarefa - Aula3

Instruções
Exercício 1 - Você está desenvolvendo um sistema de gestão para uma biblioteca. A biblioteca precisa manter o controle dos livros, dos membros e dos empréstimos.

Tarefas:

Criar Classes:

Crie uma classe Livro com os atributos titulo, autor e ISBN.
Crie uma classe Membro com os atributos nome, id e email.
Crie uma classe Emprestimo com os atributos livro, membro e dataEmprestimo.

Criar Métodos:

Implemente métodos para adicionar e remover livros da biblioteca.
Implemente métodos para registrar um novo membro.
Implemente métodos para registrar o empréstimo de um livro e para devolver um livro.

Exercício 2 - Você está desenvolvendo um sistema de pedidos para um restaurante. O restaurante precisa gerenciar os pratos do cardápio, os pedidos dos clientes e os detalhes de cada pedido.

Tarefas:

Criar Classes:

Crie uma classe Prato com os atributos nome, preco e descricao.
Crie uma classe Pedido com os atributos numeroPedido, cliente, listaDePratos e total.

Criar Métodos:

Implemente métodos para adicionar e remover pratos do cardápio.
Implemente métodos para criar um novo pedido e calcular o total do pedido.
Implemente métodos para visualizar o pedido e seus detalhes.
